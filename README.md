# SK Presidency Public School Website

![SK Presidency School](assets/images/logo/school-logo.png)

## 🎓 About This Project

SK Presidency Public School Website is a modern, responsive, and user-friendly platform designed to showcase the school's excellence in education. The website is built using **HTML5, CSS3, and JavaScript** with **Google Labs-inspired design** principles.

**CBSE Affiliation Number:** 2133231  
**Location:** Odara, Sultanpur, Uttar Pradesh  
**Founded:** 2011

---

## ✨ Features

- ✅ **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- ✅ **Modern UI** - Google Labs inspired minimalist design with smooth animations
- ✅ **Fast Performance** - Optimized for speed and SEO
- ✅ **Student-Friendly** - Easy navigation and intuitive interface
- ✅ **Interactive Elements** - Smooth scroll animations and hover effects
- ✅ **Contact Forms** - Easy communication with school administration
- ✅ **PDF Downloads** - Prospectus and admission forms
- ✅ **Social Media Integration** - Links to social media platforms
- ✅ **Mobile Navigation** - Hamburger menu for mobile devices
- ✅ **Accessibility** - WCAG compliant design
